## 🧭 What we're about

We're a small, scrappy collective that treats **constraints as a feature** — turning
limited resources into focused, self-contained software that just works.

- **🎯 Mission** — Build practical, standalone digital products (micro-apps, developer
  plugins, localized infrastructure) that solve real problems without bloat or vendor lock-in.
- **🔭 Vision** — Prove that a lean, indie-hacker approach can ship production-grade tooling.
  Every repo is a bet on craftsmanship over scale, and grit over shortcuts.
